# ToDoList

A personal Android To-Do List app built with Jetpack Compose and Clean Architecture.

## Features
- 3 task types: Habit (recurring), Daily (with roll-over), One-time (with deadline)
- Scoring system with streak multipliers
- Activity heatmap & completion dashboard
- Offline-first with optional Google account sync

## Tech Stack
Kotlin · Jetpack Compose · Room · Hilt · Firebase · WorkManager
